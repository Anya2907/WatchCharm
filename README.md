# WatchCharm
Responsive landing page for a premium watch collection.
The project demonstrates modern frontend layout techniques, responsive images, interactive UI elements and smooth animations.

## Live Demo
https://anya2907.github.io/WatchCharm/

## Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- Responsive Design
- Flexbox
- CSS Grid
- Intersection Observer API
- Swiper.js
- Formspree (contact form)

## Features
- Responsive layout for mobile, tablet and desktop
- Mobile burger menu
- Catalog "Show more" functionality
- Reviews slider implemented with Swiper.js
- Scroll to top button
- Smooth scrolling navigation
- Scroll reveal animation (Intersection Observer)
- Responsive images using `<picture>` and `srcset`
- Contact form using Formspree

## Libraries Used
- Swiper.js – slider for reviews
- Body Scroll Lock – mobile menu scroll control
- Modern Normalize – CSS normalization
